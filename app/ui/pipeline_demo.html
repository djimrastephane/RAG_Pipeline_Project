25% overlap preserves local context across chunk boundaries. Chunking uses tiktoken exact token counting during preprocessing.
04
🧠
Embedding
Sentence Transformer Encoding
all-MiniLM-L6-v2
384-dim dense vectors
Normalised for cosine sim
Lightweight 22M param model. Sufficient for semantic retrieval on structured reports; faster than larger alternatives.
🗂️
Vector Index
FAISS IndexFlatIP
Exact inner-product search
In-memory at query time
Supports hybrid BM25 + dense
IndexFlatIP is exact (no approximation). Suitable for corpus size — no need for IVF/HNSW at this scale.
💾
Persistent Storage
Embedding Vector Store
.npy per document
data_processed/doc_id/embeddings.npy
Paired with chunk metadata
Parquet for metadata, .npy for vectors. Dual-storage enables re-indexing without re-embedding.
